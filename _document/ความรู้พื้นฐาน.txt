ความรู้พื้นฐาน เทคนิค เทคโนโลยีและเครื่องมือที่ใช้
1.	ความรู้พื้นฐาน และเทคนิค OOP
		รูปแบบการทำงานของ OOP หรือ Object Oriented Programming ซึ่งมีประโยชน์ทางด่านการซ้อนดาค้าให้อยู่ในคราส ไม่ให้มีการรั่วไหลออกนอกคราสหากเราไม่ได้ทำการเรียก ป้องกันการเข้าถึงดาต้าจากบุคคลที่สามอีกด้วย อีกทั้งยังมีการสืบทอด สามารถนำมาใช้กับตัวละครซึ่งมีความสามารถที่แตกต่างกัน แต่โดยทั่วไปแล้วตัวละคร หรือศัตรูมีความสามารถบางอย่างที่มีเหมือนกัน อย่างเช่น ความสามารถในการเดิน
2.	ใช้ภาษา Java ในการพัฒนา
	โดยปกติจากการทำงานของภาษา Java มีความสอดคล้อง และเขียนได้ง่ายเมื่อต้องการเขียนโปรแกรมในลักษณะของ Object ซึ่งสอดคล้องกับความรู้พื้นฐานข้อก่อนหน้า
3.	ความรู้พื้นฐาน และเทคนิคการทำงานของ Compiler
		หลักพื้นฐานของ compiler high language programming คือการนำสิ่งที่ผู้ใช้พิมพ์มาแปลงให้เป็น machine code แล้วคอมพิวเตอร์จึงทำงานจาก machine code นั้น ซึ่งจากโครงงานนี้ได้หยิบลักษะของ compiler ทีมีการแปลงภาษาระดับสูงให้เป็นภาษาที่คำสั่งภายในเกมเข้าใจ
